کورس های رایگان ما

Hypertext Markup Language

زبان HTML، پایه و اساس صفحات وب را تشکیل میدهد و اولین قدم برای شروع طراحی صفحات وب محسوب میشود.

بیشتر بیاموزید

Cascading Stylesheet CSS

CSS مخفف Cascading Style Sheets به معنای برگه های سبک آبشاری است. CSS یک زبان شیوه نامه استاندارد است که برای توصیف نمایش (یعنی طرح و قالب بندی) صفحات وب استفاده می شود.

بیشتر بیاموزید

javaScript Language

جاوااسکریپت (JavaScript) یک از زبان برنامه نویسی شیء گرا و پرطرفدار وب میباشد. این زبان را در ابتدا شخصی به نام Brendan بعدا به LiveScript و Eich (برندان ایچ) در شرکت Netscape با نام Mocha طراحی نمود.
